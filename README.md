# SIGI Backend

Backend del **Sistema Inteligente de Gestión de Inventario (SIGI)** desarrollado con **Spring Boot**.  
Provee APIs REST seguras, comunicación en tiempo real vía WebSocket STOMP y persistencia con **Postgres/MySQL**.  
Incluye autenticación con **JWT**, migraciones con **Flyway** y pruebas automatizadas.

---

## 🚀 Tecnologías principales

- **Java 17+**
- **Spring Boot 3.x**
- **Spring Security + JWT**
- **Spring Data JPA**
- **WebSocket STOMP**
- **Flyway** (migraciones)
- **Postgres/MySQL**
- **JUnit 5 + Mockito + Testcontainers**
